# 📰 AI-Powered Tamil News Summarizer

A real-time news summarizer built using Groq's LLaMA model and Streamlit that summarizes any news article in English or Tamil using prompt engineering.

## 🚀 Live Demo
👉 [tamil-news-summarizer.streamlit.app](https://tamil-news-summarizer.streamlit.app)

## 🛠️ Tech Stack
- Python
- Groq API (LLaMA 3.3)
- Streamlit
- Prompt Engineering
- python-dotenv

## ✨ Features
- Paste any news article in English or Tamil
- Get a clean 5-point AI summary instantly
- Choose summary language (English or Tamil)
- Secure API key handling

## 🔧 How to Run Locally
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add: `GROQ_API_KEY=your_key_here`
4. Run: `streamlit run app.py`
